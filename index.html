About FitForge

Purpose

FitForge was engineered to remove the friction from workout planning. By taking in specific user metrics, it calculates a structured, balanced 7-day routine tailored to exact goals and limitations.

Technologies

Built with a modern web stack focusing on performance and encapsulation:

  • HTML5: Semantic, accessible application structure.
  • CSS3: Custom properties, CSS Grid/Flexbox architecture, responsive design.
  • Vanilla JavaScript (ES6+): State management, DOM manipulation, and dynamic data rendering without bloated frameworks.
  • Local Storage API: Persistent data tracking across sessions.

How it Works

The application relies on a comprehensive internal array of exercise objects. When you submit your parameters, the generator's algorithm filters the master database based on your equipment and experience, then structures those exercises into a logical 7-day split (Push/Pull/Legs, Upper/Lower, or Full Body depending on the inputs).
